# 💘 Interactive "Do You Love Me?" Mini-Project

### A playful and interactive web project featuring an evasive button!

## 📌 Project Overview
This is a fun, lighthearted front-end mini-project built to practice DOM manipulation and interactive CSS. It asks a simple question: "Do you love me?", but with a twist! The "No" button actively dodges the user's cursor, making it impossible to click anything other than "Yes". 

Once the user finally clicks "Yes", a rewarding animation plays!

## 🚀 Key Features
- **Evasive Button Mechanics:** The "No" button uses JavaScript to dynamically change its position and run away from the mouse hover.
- **Interactive Media:** Integrates embedded autoplaying videos/GIFs that react based on the user's choices.
- **Clean UI:** A simple, centered, and aesthetically pleasing user interface.

## 🛠️ Tech Stack
- **HTML5 & CSS3:** For the core structure, styling, and layout.
- **Vanilla JavaScript:** For handling the DOM events and the mathematical logic behind the dodging button.
